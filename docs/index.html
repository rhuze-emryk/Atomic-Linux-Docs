What are Atomic Linux Distributions?

Atomic (or immutable) Linux distributions feature a read-only operating system core that remains consistent and unchanged during normal operation. Changes to the system are made atomically - meaning they're applied all at once during controlled updates or reboots.

Popular atomic distributions include:

  • Fedora Silverblue/Kinoite/Sericea
  • openSUSE MicroOS
  • NixOS
  • Vanilla OS
